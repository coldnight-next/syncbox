Under the hood

The technology behind Syncbox

Built on proven protocols and modern cryptography for reliable, secure file synchronization.

Peer-to-Peer Architecture

Direct encrypted connections between devices. Files transfer over LAN at full speed, or via relay when on different networks.

mDNS Discovery

Automatic device discovery using multicast DNS. No manual IP configuration needed. Devices find each other instantly on your network.

Vector Clocks

Distributed timestamps track every file change. Syncbox detects concurrent edits and resolves conflicts without data loss.

Ed25519 Encryption

All data in transit is encrypted using Ed25519 device identity keys. Your files are never readable by anyone other than your own devices.
