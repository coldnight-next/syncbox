Why Syncbox

Everything you need to stay in sync

Stop worrying about which device has the latest version. Syncbox handles it all.

Instant sync

Changes appear on all devices within seconds. No waiting for cloud uploads — files transfer directly between your machines.

End-to-end encrypted

Every file is encrypted before it leaves your device. Only your paired devices can decrypt them. We never see your data.

No storage limits

Sync as many files as you want. Since files go directly between devices, there's no cloud storage quota to worry about.

Smart conflict resolution

Edit the same file on two devices? Syncbox detects the conflict and lets you choose which version to keep — or keep both.

Runs silently

Lives in your system tray, starts with your PC, and syncs in the background. You'll forget it's there — until you need it.

Peer-to-peer

Your files never touch a third-party server. Devices discover each other on your network and sync directly. Your data stays yours.

1

Install Syncbox

Your files never touch our servers. Syncbox uses mDNS to discover other devices on your local network and establishes direct, encrypted WebSocket connections between them. All file data is transferred peer-to-peer.

Syncbox uses vector clocks to detect conflicts. When it finds one, you can choose to keep the local version, the remote version, or both (it creates a conflict copy). You can also set it to auto-resolve using last-writer-wins.
